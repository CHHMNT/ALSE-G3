README:
El programa de C++ calcularMayorMagnitud permite al ussuario ingresar un numero en un plano con cordenas x & y.
El programa claculara la magnitud de cada punto y determinar cual de los diferentes puntos tiene la mayor magnitud regresando magnitud y el indice del punto.
Se usa un estructura Point para representar puntos, y la funcion principal main() maneja la entrada y salida de datos.
